Incrustation PHP dans une Image JPEG by Trhacknon - Guide d'utilisation
===============================================================

Ce script permet d'injecter du code PHP malicieux dans une image JPEG.
Assurez-vous d'utiliser cet outil à des fins éducatives et légales seulement.


ÉTAPES D'UTILISATION :

1. Assurez-vous d'avoir Python 3.x installé sur votre système.

2. Placez l'image JPEG d'origine que vous souhaitez infecter dans ce répertoire.

3. Ouvrez un terminal et accédez au répertoire contenant ce script.

4. Exécutez le script en entrant la commande suivante :
   phptojpeg

5. Suivez les instructions à l'écran pour effectuer les étapes suivantes :
   - Entrez le chemin de l'image JPEG d'origine.
   - Entrez le code PHP malicieux que vous souhaitez injecter.
   - Entrez le chemin de sortie pour l'image infectée.

6. Une fois que le processus est terminé, l'image infectée sera enregistrée au chemin que vous avez spécifié.

REMARQUES :

- Assurez-vous d'avoir installé le module "colorama" en exécutant la commande suivante :
   pip install colorama

- N'utilisez pas cet outil à des fins malveillantes. Respectez les lois en vigueur.
